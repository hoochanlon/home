# HomePage

> [!important]
> 源项目：https://github.com/SimonAKing/HomePage

一个现代优雅的个人主页，具有流体动画背景、响应式设计和流畅的页面过渡效果。

## 安装步骤

```sh
git clone https://github.com/SimonAKing/HomePage.git
cd HomePage
npm install
npm run dev
```
